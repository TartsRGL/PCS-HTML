# Lesson 3 – Layout Containers (HTML View)

This folder contains the **HTML-only code** for Lesson 3 of the **PCS-HTML** series.

This lesson introduces **layout containers at the HTML level** before any CSS layout system (Flexbox or Grid) is applied.

The goal is to understand that layout behavior starts with structural decisions in HTML.


It demonstrates:

- Difference between semantic elements and layout wrappers
- Why a global page wrapper may exist
- Why content containers are separated from layout wrappers
- How grouping elements affects future CSS layout
- How nested layout contexts are created
- When an extra `div` is structurally justified

This lesson prepares HTML so CSS can react correctly later.

Each commit introduces **one structural layout decision at a time**.
