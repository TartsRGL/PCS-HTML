# Lesson 1 – HTML Skeleton & Semantics

This folder contains the **HTML-only code** for Lesson 1 of the **PCS-HTML** series.

It focuses on building a **correct HTML foundation** before any CSS or layout decisions are made.

It demonstrates:

* Basic HTML document structure (`<!doctype>`, `html`, `head`, `body`)
* Global semantic regions: `header`, `main`, `footer`
* Content grouping with `article` and `section`
* Page-level vs article-level semantics
* Why multiple `header` / `footer` elements are valid

Each commit introduces **one clear HTML decision at a time** – step-by-step.

## 📌 Commits overview

```bash
feat: add base HTML skeleton
feat: add basic metadata
feat: add global semantic regions
feat: add page header content
feat: add main article structure
feat: add internal article semantics
feat: add global footer content
chore: final semantic sanity check
```
