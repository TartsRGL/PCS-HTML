
# Lesson 8 – Real World Composition

This folder contains the **HTML-only code** for Lesson 8, the grand finale of this HTML series!

It focuses on bringing together all the concepts learned throughout the previous 7 lessons: semantic skeleton, content hierarchy, foundational layout containers, media, components, and forms – combining them into a full page structure based on a generic "Tech Blog" scenario.

It demonstrates:

-**Document Scaffolding**: Structuring `<header>`, `<main>`, and `<footer>` consistently.

-**Skip Links & A11y basics**: Including high-level accessibility features for keyboard users.

-**The Article & Aside Combo**: Creating a strong text hierarchy with `<article>`, `<section>`, and pairing it with related sidebar content `<aside>`.

-**Form Integration**: Utilizing the strict semantic rules established in Lesson 7 within a larger document context.

-**The "No-Div" Challenge**: Keeping the `<div>` tag count to an absolute minimum, reserving it strictly for CSS grid/flexbox helper containers where semantic tags aren't applicable.

Each commit introduces **one clear structural evolution** – layering the page logically step-by-step.
